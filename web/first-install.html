Default starter

humanizer

Claude Code skill that removes signs of AI-generated writing from text.

git clone https://github.com/blader/humanizer.git ~/.claude/skills/humanizer
  • Fit: Best for first-time visitors who want one clear, low-friction install with fast visible payoff.
  • Caution: Low to medium risk; the main question is whether the output style fits your writing workflow.
  • Why this is a first-install pick: Single-purpose, clear command, low surprise, fast feedback.
