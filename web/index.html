Skill Tree · Decision layer before install

Skill Tree is not another awesome list — it is the decision layer before install.

Most lists help you discover that a skill exists. Skill Tree helps you decide whether it is worth installing, how to install it, what risk it adds, and what kind of agent workflow it actually belongs to.

A first-time visitor should be able to understand a skill’s value in about 30 seconds: what it does, how to install it, why it is risky or safe enough, and whether it fits their workflow.

Why not another awesome list?

What users need to know fast

  • What problem does this skill solve?
  • Who should try it first?
  • How do I install it?
  • What risk am I taking?
